# Docker Configuration

Docker and container-related configurations.

## Files

- Docker daemon configuration
- Container templates
- Docker Compose configs

## Usage

These configurations are typically symlinked to `~/.config/docker/` or similar locations.

## Dependencies

- Docker
- Docker Compose (optional)

## Common Commands

```bash
docker ps          # List containers
docker images      # List images
docker-compose up  # Start services
```

## Note

This directory may contain service-specific Docker configurations. Check individual files for details.
