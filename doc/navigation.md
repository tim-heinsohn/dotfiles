# Navigation Index

Quick reference for finding things in this repository.

## By Topic

### Shell & Terminal
- Zsh config: `zsh/`
- Bash config: `shell/`
- Terminal: `wezterm/`, `alacritty/` (if exists)

### Editor
- Neovim: `nvim/`
- Vim: `vim/`

### Development
- Git: `git/`
- Ruby/Rails: `rails/`, `ruby/`
- Node.js: `npm/`
- Terraform: `terraform-ls/`

### Desktop
- i3 window manager: `i3/`
- Rofi: `rofi/`
- X11: `xinit/`, `xmodmap/`

### Security
- GPG: `gpg/`
- GoPass: `gopass/`

### Scripts & Tools
- Bin scripts: `bin/`
- Utils: `utils/`

## Quick Lookup

| Need | Location |
|------|----------|
| Add new dotfile | Edit `bombadil.toml` |
| Find symlink target | Check `bombadil.toml` [settings.dots] |
| Edit shell alias | `zsh/aliases` |
| Edit git alias | `git/gitconfig` |
| Edit editor config | `nvim/` or `vim/` |
| Run system setup | `bin/setup-system` |
| Manage packages | `bin/packages` |
