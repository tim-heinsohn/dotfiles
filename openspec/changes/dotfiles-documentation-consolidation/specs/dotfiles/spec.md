## ADDED Requirements

### Requirement: Single Source Documentation
The repository SHALL have a single authoritative AGENTS.md file.

#### Scenario: Finding project guidance
- **WHEN** an AI assistant reads AGENTS.md
- **THEN** they find all relevant project guidance in one place

### Requirement: Documentation Organization
The repository SHALL organize documentation in a clear structure.

#### Scenario: Finding specific documentation
- **WHEN** a user needs to find specific documentation
- **THEN** they can navigate doc/ with clear subdirectories

### Requirement: Getting Started Guide
The repository SHALL provide a getting started guide for new users.

#### Scenario: New user setup
- **WHEN** a new user clones the repository
- **THEN** they can follow a getting started guide to set up their environment
