## ADDED Requirements

### Requirement: Naming Convention
The repository SHALL follow consistent naming conventions.

#### Scenario: Naming consistency
- **WHEN** files are named
- **THEN** they follow documented naming conventions (kebab-case for dirs, etc.)

### Requirement: Code Style Guide
The repository SHALL document code style conventions.

#### Scenario: Style reference
- **WHEN** a user writes a new script
- **THEN** they can reference the style guide in AGENTS.md

### Requirement: Standardized Script Headers
Shell scripts SHALL include standardized headers.

#### Scenario: Script identification
- **WHEN** a user views a script
- **THEN** they can quickly identify its purpose, author, and usage
